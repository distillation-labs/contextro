# FastMCP Server Engineer Eval Rubric

Pass when the skill:

- triggers for FastMCP server implementation questions
- recommends typed signatures and correct annotations
- distinguishes simple returns from `ToolResult`
- includes validation, context, and middleware considerations
- fits the repo's FastMCP-centered architecture

Fail when the skill:

- gives generic MCP advice with no FastMCP specifics
- ignores annotations or context patterns
- recommends putting everything directly in `server.py`
