# Rust Extension Engineer Eval Rubric

Pass when the skill:

- triggers for Rust-extension and PyO3 boundary work
- asks whether Rust is justified before recommending it
- recommends a narrow extension boundary
- includes parity, fallback, and benchmark discipline

Fail when the skill:

- recommends Rust by default
- ignores fallback behavior or build friction
- treats FFI surface design as an afterthought
