# Python Systems Engineer Eval Rubric

Pass when the skill:

- triggers for Python code quality and refactor questions
- recommends precise typing and explicit boundaries
- includes exceptions or concurrency when relevant
- favors minimal, testable refactors

Fail when the skill:

- defaults to `Any`
- gives framework-specific FastMCP advice when the issue is generic Python design
- recommends abstraction without a concrete readability or correctness gain
